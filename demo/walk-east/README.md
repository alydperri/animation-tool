# Walk-East Demo

This demo shows a completed `walk-east` animation created from one character
anchor image.

- `cat-identity.png`: source identity anchor
- `walk-east.gif`: approved animated preview
- `walk-east-spritesheet.png`: final transparent spritesheet

Runtime sheet details:

- frame count: 8
- grid: 4 columns x 2 rows
- frame size: 256 x 256 px
- sheet size: 1024 x 512 px
