# Jump-2 Demo

This demo shows a completed presentation GIF workflow created from one spider
anchor image.

- `spider-identity.png`: source identity anchor
- `jump-2.gif`: approved looping animated preview
- `jump-2-spritesheet.png`: final transparent spritesheet

This example uses the x-only normalization path:

- horizontal drift is corrected
- vertical jump motion is preserved in the GIF
- selected frames were removed during review to improve the loop

Runtime sheet details:

- frame count: 19
- grid: 5 columns x 4 rows
- empty cells: 1 transparent slot
- frame size: 256 x 256 px
- sheet size: 1280 x 1024 px
